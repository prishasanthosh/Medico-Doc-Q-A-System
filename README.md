# 🩺 Medico Doc Q&A with Gemini

A Document Question Answering system that answers the medical professionals to answer their queries from the uploaded medical documents. Powered by Google Gemini, LangChain, and Chroma VectorDB.

## Features

- Extracts and understands content from medical PDFs
- Uses Google Generative AI embeddings for semantic search
- Stores and retrieves document chunks using Chroma Vector Database
- Answers questions using Gemini 1.5 Pro via LangChain's RetrievalQA
- Simple Gradio-based UI
